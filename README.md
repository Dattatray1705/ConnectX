# ConnectX

A full-stack LinkedIn-inspired social networking platform built with **Next.js**, **Node.js**, **Express.js**, **MongoDB**, **Redux Toolkit**, **JWT Authentication**, and **Cloudinary**.

## ✨ Features

- User Authentication (JWT)
- Profile Management
- Create, Like & Comment on Posts
- Discover Users
- Image Upload
- Protected Routes
- Responsive UI

## 🛠 Tech Stack

- **Frontend:** Next.js, React, Redux Toolkit
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Authentication:** JWT
- **Storage:** Cloudinary

## 🚀 Getting Started

```bash
# Backend
cd backend
npm install
npm run dev

# Frontend
cd frontend
npm install
npm run dev
```
